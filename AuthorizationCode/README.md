# Authorization Code Demo

## Prepare

Create an OAuth client at github.

### GitHub

1. [https://github.com/settings/developers](https://github.com/settings/developers)
2. New OAuth App
3. Homepage-url: http://localhost:8080
4. Authorization callback URL: http://localhost:8080/github/login
5. Add credentials to src/server.js

## Install

```cmd
npm install
```

## Run

```cmd
npm start
```

## Demo

```browser
localhost:8080/
```
